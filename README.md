# 寝室厨房打扫系统

这是一个为寝室厨房打扫管理设计的Web应用程序，支持用户版和管理员版，数据通过浏览器本地存储共享。

## 功能特性

### 用户版
- 选择自己的房间号
- 查看今日打扫房间
- 决定是否永久跳过打扫
- 轮到打扫时拍照上传记录
- 查看所有房间状态和历史记录

### 管理员版
- 手动设置今日打扫房间
- 强制管理房间状态（跳过/重新加入）
- 重置所有数据
- 导入/导出数据
- 查看打扫照片

## 部署说明

### 通过GitHub Pages部署（推荐）

1. 在GitHub上创建一个新的仓库
2. 将所有文件上传到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择分支（通常是main）和根目录
5. 访问生成的网址（例如：`https://你的用户名.github.io/仓库名`）

### 本地部署

1. 将整个文件夹放在Web服务器上
2. 通过浏览器访问 `index.html`

## 使用说明

### 用户使用
1. 访问主页面
2. 点击"用户版"进入用户界面
3. 选择自己的房间号
4. 查看信息和操作

### 管理员使用
1. 访问主页面  
2. 点击"管理员版"进入管理员界面
3. 使用各种管理功能

## 技术栈

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (ES6+)
- 浏览器本地存储 (localStorage)

## 注意事项

- 数据存储在浏览器本地，清除浏览器数据会丢失所有记录
- 建议定期导出数据备份
- 系统使用拍照功能，需要设备支持摄像头
